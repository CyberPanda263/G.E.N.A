## [4.0.0] — 2026-05-23
### 🚀 Додано
* **Нова архітектура моніторингу:** Повна інтеграція зв'язки Prometheus + Loki для збору метрик та логів в одному місці (Grafana).
* **Signal Alerting Adapter:** Створено кастомний Python-адаптер для інтеграції `Alertmanager` з месенджером Signal через офіційну двофакторну авторизацію (QR-код у терміналі).
* **Enterprise Hardening (Безпека):** Автоматичне налаштування брандмауера UFW (Default Deny), оптимізація параметрів ядра `sysctl` для Kubernetes та встановлення Fail2Ban проти brute-force атак на SSH.
* **Інтерактивне меню встановлення:** Додано вибір режимів Let's Encrypt (Staging/Production) та типів розгортання Portainer (Agent/Server).

### ⚙️ Змінено
* **Міграція на K3s:** Повністю оновлено базовий рушій кластера на легковажний дистрибутив K3s версії `v1.31.6+k3s1` замість "важкого" ванільного Kubernetes.
* **GitOps за замовчуванням:** Панель ArgoCD тепер розгортається автоматично з увімкненим замовчуванням механізмом `selfHeal` та `prune`.
* **SSL через DNS-01:** Cert-Manager переведено на автоматичну перевірку челенджів через Cloudflare API Токени, що дозволяє випускати сертифікати для закритих/локальних хостів.

### 🔒 Безпека
* Вимкнено автентифікацію по паролях для SSH (`PasswordAuthentication no`).
* Заборонено прямий логін для користувача `root` за паролем.
